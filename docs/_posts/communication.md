### Komunikacija

```yml

Nacini: miran ton, vikanje, signalizacija, prenosenje poruke preko ljudi

Strane: starboard, port 

Redosled: Skiper pa kormilar (bez demokratije)

```

```yml
VHF: koriscenje

1- DSC (5 sec) =>(15 sec) repeat on CH 16 VHF
2- Mayday Mayday Mayday (or Pan Pan, Pan Pan Pan Pan)
3- position
```

```yml
Distress-alerting:
- flags NC
- letter V (morse, flag)
- red hand flare / orange smoke / rocket
- SOS,  by any means
- square flag + ball (over or under)
- continious fog horn
- continious arm signals
```

```yml
Helpful-behaviour: ponavljanje primanja komande, obavestavanje o izvrsenom 

Helpful-behaviour: Obavestavanje kormilara i skipera na potencijalnu opasnost (collision course, gear issues...)  / + bolje da svako kaze istu stvar, nego da svi misle da je vec neko drugi mozda rekao/

Helpful-behaviour: Obavestavanje skipera o nepoznatom (zvuk rada vodene pumpe...)
```

```yml
Sidrenje: pokazati dogovorenu signalizaciju
```
